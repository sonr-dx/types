Query

Query provides defines the gRPC querier service.

Method NameRequest TypeResponse TypeDescription
Params QueryRequest QueryParamsResponse

Params queries all parameters of the module.
Accounts QueryRequest QueryAccountsResponse

Accounts returns associated wallet accounts with the DID.
Credentials QueryRequest QueryCredentialsResponse

Credentials returns associated credentials with the DID and Service Origin.
Resolve QueryRequest QueryResolveResponse

Resolve queries the DID document by its id.
Service QueryRequest QueryServiceResponse

Service returns associated ServiceInfo for a given Origin
Token QueryRequest QueryTokenResponse

Token returns the current authentication token for the client.

Methods with HTTP bindings

Method Name Method Pattern Body
Params GET /did/params
Accounts GET /did/{did}/accounts
Credentials GET /service/{origin}/{subject}/credentials
Resolve GET /did/{did}
Service GET /service/{origin}
Token POST /token

RegisterService initializes a Service with a given permission scope and URI. The domain must have a valid TXT record containing the public key.
